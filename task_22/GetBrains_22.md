### Алгоритм мутации

#### Входные данные

- `особь`
- `максимальный_вес`
- `предметы`

#### Шаги алгоритма

1. Случайным образом выбрать позицию `i` в особи.
2. Если `особь[i] == 1`, то:
    1. Заменить `особь[i]` на `0` (убрать).
    2. Вычислить новый вес особи.
    3. Если новый вес не превышает `максимальный_вес`, то:
        - Принять изменение.
    4. Иначе:
        - Отменить изменение.
3. Иначе, если `особь[i] == 0`, то:
    1. Заменить `особь[i]` на `1` (добавить).
    2. Вычислить новый вес особи.
    3. Если новый вес не превышает `максимальный_вес`, то:
        - Принять изменение.
    4. Иначе:
        - Отменить изменение.


**Замечание**: Проводить мутацию для 5-й особи не имеет смысл, т.к вес на пределе - любой предмет превысит лимит. 
Делаем мутацию для 6-й особи. В других задачах с применением генотипа можно делать полностью рандомный выбор особи. В нашем
случае ещё необходимо сделать проверку на релевантность. Для 6-й особи имеет смысл только мутация с 1-м предметом
