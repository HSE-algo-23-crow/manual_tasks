## Задание
Для каждого варианта представлены условия для двух задач. Для каждой задачи необходимо: 
1. Выбрать алгоритм решения задачи и обосновать свой выбор.

## Задача 2:

$$T_{max} = 8$$
$$T_{avg} = (4+4+8+7+6+7+4) / 4 = 10$$

$$T_{opt} = max(8, 10) = 10$$
